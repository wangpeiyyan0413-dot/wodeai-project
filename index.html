AI INTERACTIVE SYSTEM

Body Language Emotion Recognition

探究身体姿态与情感表达的数字化联系

Project Description

This project is a real-time interactive system based on machine learning that recognizes human emotions through body movements and posture. Unlike traditional methods, we focus on body language as the primary medium of expression.

How to Use

😐 Calm

Neutral expression, arms hang naturally, relaxed still posture.

Keywords: still, neutral

😰 Anxious

Face on hand, holding head, leaning forward, or restless movements.

Keywords: tense, inward

😊 Happy

Raising arms, dancing, tilting head upward, energetic movements.

Keywords: open, energetic

💡 Tips: Make sure you are in a well-lit environment and alone in the frame.
